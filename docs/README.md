Fitur penting yang sudah didukung:

- compile-time `import "./file.haptic"`
- `export func ...`
- output `esm` dan `cjs`
- `config.hpconf` dengan metadata `package`

Urutan baca yang disarankan:

1. Mulai dari tutorial sesuai use case Anda.
2. Pakai syntax reference saat mulai menulis flow yang lebih besar.
3. Lihat folder `examples/` untuk contoh tambahan yang lebih kecil.
